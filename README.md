# Liam Andell — Portfolio Website

## Overview
This repository contains the source code for my personal portfolio website, built to showcase my experience in instructional design, training delivery, and user adoption. The site highlights my professional work at Verdtek as well as my background in teaching and facilitation.

The goal of this portfolio is to demonstrate my ability to:
- Design and develop modern web-based learning experiences
- Translate complex systems into clear, user-friendly instruction
- Build structured training content that supports real-world performance
- Combine instructional design with technical implementation

## Local Development

To run this project locally:

```bash

```

Then open:

http://localhost:5173/

---

## Build and Deployment

### Build the project

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

This project uses the `gh-pages` package to deploy the built `/dist` folder to the `gh-pages` branch.
